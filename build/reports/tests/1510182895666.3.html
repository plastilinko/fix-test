Шаг "Результат".

На данном шаге, пользователю в табличном виде представляются данные, которые он ввел на шаге "Ввода данных", а также строку с результатом вычисления. Строка с результатами должна содержать результат сложения трех чисел, которые ввел пользователь на первом шаге. Если результатом операции является четное число, последняя строка должна быть зеленого цвета.

Результат

Значение 1 12345678901234567890
Значение 2 1
Значение 3 1
Итого: 12345678901234567000
